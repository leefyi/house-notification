# CONTEXT.md

## 领域术语

- **保租房**（保障性租赁住房）：深圳市政府面向新市民、青年人提供的租赁型保障住房，租金低于市场价。通过认租申请获取。
- **公租房**（公共租赁住房）：深圳市政府面向中低收入住房困难家庭提供的租赁型保障住房。通过轮候+认租申请获取。
- **安居型商品房**：深圳市政府提供的可买卖型保障住房（本项目不监控）。
- **认租**：对租赁型保障住房（保租房/公租房）提交租赁申请的行为。
- **轮候**：公租房申请审核通过后进入排队序列，等待分配的状态。
- **通知公告**：深圳市住建局网站发布的最新住房保障相关公告，包含配租通告、选房通知等。
- **Bark**：iOS 推送通知 App，通过 device key 发送推送。
- **已见 URL**：程序已经抓取并记录过的公告链接，用于去重避免重复通知。

## 业务规则

- 公告来源：深圳市住房和建设局网站（zjj.sz.gov.cn）通知公告列表页
- 通知条件：公告标题包含"保租房"或"公租房"或"认租"相关关键词
- 通知内容：公告标题 + 原文链接
- 监控频率：每日 3 次（09:00 / 13:00 / 17:00 CST）
- 区域范围：深圳全市，不做区级过滤
